# Operadores

En esta opción se podrá configurar a detalle cada operador que haga parte del grupo empresarial.

## Ingreso a la opción

Para acceder, debes ingresar a la opción de “Operadores”:
<br />
<div className="image-zoom-wrapper">
  <Zoom>
    <Image
      src="/opcionOperadores.webp"
      alt="Opción operadores"
      width={700}
      height={300}
      unoptimized
    />
  </Zoom>
</div>

## Crear operador   

Dentro de la opción de "Operador", se encuentra disponible un botón con el texto "Nuevo operador". 
<br />
<div className="image-zoom-wrapper">
  <Zoom>
    <Image
      src="/nuevoOperador.webp"
      alt="Nuevo operador"
      width={700}
      height={300}
      unoptimized
    />
  </Zoom>
</div>

Al hacer clic en este botón, se abrirá el siguiente formulario que permite la entrada de datos para la creación de un nuevo operador:

- **Tipo Identificación (Campo de selección, Obligatorio).**
- **Número de identificación(Obligatorio).**
- **Primer nombre(Obligatorio).**
- **Segundo Nombre.**
- **Primer apellido (Obligatorio).**
- **Segundo apellido.**
- **Sexo (Obligatorio).**
- **Fecha de nacimiento(Obligatorio).**
- **Teléfono (Obligatorio).**
- **Correo electrónico (Obligatorio).**
- **Contraseña (Obligatorio).**
- **Sede (Campo de búsqueda, obligatorio).**

<br />
<div className="image-zoom-wrapper">
  <Zoom>
    <Image
      src="/formularioOperador.webp"
      alt="Formulario operador"
      width={700}
      height={300}
      unoptimized
    />
  </Zoom>
</div>

## Buscar operador

Dentro de la opción, el campo de búsqueda permite listar operadores  a partir de un nombre, teléfono o correo que se digite:

## Asignar permisos 

En el formulario de edición en la pestaña “Permisos”, se gestionan los permisos que va a tener cada operador:

* [crear permisos](/moduloAdministrador/seguridad/permisos): en este link podrá aprender a crear permisos.

<br />
<div className="image-zoom-wrapper">

### Búsqueda
En este campo podrá buscar por descripción los permisos que desee asignar al operador, una vez desplegados los resultados selecciona el permiso y el sistema lo asigna automáticamente:
