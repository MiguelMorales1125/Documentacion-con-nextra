# Operadores

En esta opción se podrá configurar a detalle cada operador que haga parte del grupo empresarial.

## Ingreso a la opción

Para acceder, debes ingresar a la opción de “Operadores”:
<br />

  <img
    src="/opcionOperadores.webp"
    alt="Opción operadores"
    style={{ width: '100%', borderRadius: '8px' }}
  />    
</div>

## Crear operador   

Dentro de la opción de "Operador", se encuentra disponible un botón con el texto "Nuevo operador". 
<br />
<div style={{ maxWidth: '700px', margin: '0 auto' }}>
  <img
    src="/nuevoOperador.webp"
    alt="Nuevo operador"
    style={{ width: '100%', borderRadius: '8px' }}
  />    
</div>

Al hacer clic en este botón, se abrirá el siguiente formulario que permite la entrada de datos para la creación de un nuevo operador:

- **Tipo Identificación (Campo de selección, Obligatorio).**
- **Número de identificación(Obligatorio).**
- **Primer nombre(Obligatorio).**
- **Segundo Nombre.**
- **Primer apellido (Obligatorio).**
- **Segundo apellido.**
- **Sexo (Obligatorio).**
- **Fecha de nacimiento(Obligatorio).**
- **Teléfono (Obligatorio).**
- **Correo electrónico (Obligatorio).**
- **Contraseña (Obligatorio).**
- **Sede (Campo de búsqueda, obligatorio).**

<br />
<div style={{ maxWidth: '700px', margin: '0 auto' }}>
  <img
    src="/formularioOperador.webp"
    alt="Formulario operador"
    style={{ width: '100%', borderRadius: '8px' }}
  />    
</div>
