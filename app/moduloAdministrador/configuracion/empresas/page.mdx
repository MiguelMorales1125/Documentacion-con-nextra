# Empresas

En esta opción se podrá configurar a detalle cada empresa que haga parte del grupo empresarial.

## Ingreso a la opción 
Para acceder, debes ingresar a la opción de “Empresas”:
<br />
<div className="contenedor-imagenes">
  <ImageZoom
    src="/empresas.webp"
    alt="Apartado empresas"
    width={800}
    height={300}
  />
</div>

## Crear empresa

Dentro de la opción de **Empresas**, se encuentra disponible un botón con el texto **Nueva empresa**. 
<br />
<div className="contenedor-imagenes">
  <ImageZoom
    src="/nuevaEmpresa.webp"
    alt="Nueva empresa"
    width={800}
    height={300}
  />
</div>

Al hacer clic en este botón, se abrirá el siguiente **formulario** que permite la entrada de datos para la creación de una nueva empresa:

- **Tipo Identificación (Obligatorio).**
- **Número de identificación (Obligatorio).**
- **Dígito de verificación (Obligatorio).**
- **Naturaleza jurídica (Campo de selección, Obligatoria):** Puede seleccionar alguna de estas opciones:
    - Privada
    - Mixta
    - Pública
- **Nombre o razón social (Obligatorio).**
- **Teléfono (Obligatorio).**
- **Dirección (Obligatorio).**
- **Barrio (Obligatorio).**
- **Municipio (Campo de selección, obligatorio).**
- **Fax (Obligatorio).**
- **Email (Obligatorio):** Este es un campo único
- **Representante legal (Obligatorio).**
- **Dominio (Obligatorio):** Dirección URL de dominio.
- **Logo (Obligatorio):** URL tipo http en el cual está alojado el logo que usará cada empresa.

<br />
