# Sedes 

Las sedes se refieren a las ubicaciones físicas donde se encuentran las instalaciones médicas, como consultorios, clínicas u hospitales. Cada sede puede tener su propia dirección, información de contacto. **La información tratada en esta opción será compartida entre los módulos de citas, administrador y digiturno.**

## Ingreso a la opción

1.  Para acceder, primero debe ingresar a la opción de configuración desde el menú principal del módulo de citas.
2.  Una vez en la configuración, se desplegará un submenú donde encontrará la opción específica "Sedes". Al seleccionar esta opción, podrá gestionar y administrar las sedes donde se atenderá a los usuarios.

## Crear sede

Dentro de la opción de "Sedes", se encuentra disponible un botón con el texto "Nueva sede". 

(foto del botón nueva sede)

Al hacer clic en este botón, se abrirá el siguiente formulario que permite la entrada de datos para la creación de una nueva sede:

- **Código (Campo único y obligatorio):** Este campo requiere un código único para identificar de manera exclusiva la nueva sede dentro del sistema.
- **Descripción:** En este campo, se debe proporcionar una descripción concisa pero completa del grupo de servicio. Esta descripción ayudará a contextualizar y comprender mejor la función y el propósito del grupo de servicio dentro del sistema.
- **Nombre (Obligatorio):** Aquí se debe ingresar el nombre de la sede, proporcionando una identificación clara y descriptiva.
- **Dirección (Obligatoria):** Este campo requiere la dirección física de la sede.
- **Barrio (Obligatorio):** Se debe especificar el barrio donde está ubicada la sede.
- **Municipio (Campo de búsqueda, obligatorio):** Permite buscar y seleccionar el municipio al que pertenece la sede, asegurando una asignación correcta.
- **Zona (Campo de selección, obligatorio):** Este campo ofrece la opción de seleccionar entre "Rural" o "Urbano", clasificando la ubicación de la sede.
- **Teléfono (Obligatorio):** Se debe ingresar el número de teléfono de contacto de la sede.
- **Correo Electrónico (Campo obligatorio):** En este campo, se debe proporcionar la dirección de correo electrónico de la sede.
- **Gerente (Obligatorio):** Aquí se debe ingresar el nombre del gerente encargado de la sede.
- **Permitir pacientes no registrados (Funcionalidad para digiturno, Switch):** para controlar si los pacientes que no estén registrados en las aseguradoras asociadas a la sede puedan ser atendidos.
- **Hora inicio (Campo con formato de hora):** Será la hora a partir de la cual se empezará a dar servicio al usuario.
- **Hora final (Campo con formato de hora):** Será la hora a partir de la cual se terminará la atención de usuarios.
- **Límite de turnos (Campo numérico):** Cantidad máxima diaria de turnos que podrá tener un paciente registrado, si este campo es cero (0), los usuarios no tendrán limitaciones en el número de turnos.
