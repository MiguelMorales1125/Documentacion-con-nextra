# Webhooks

En esta opción se podrá configurar los webhooks para la migración de información.

## Ingreso a la opción 
Para acceder, debes ingresar al submenú “Integraciones”, luego la opción “Webhooks”.
<br />

  <ImageZoom
    src="/webhooks.webp"
    alt="Apartado webhooks"
    width={800}
    height={300}
  />
</div>

## Crear webhook

Dentro de la opción de "Webhooks", se encuentra disponible un botón con el texto "Nuevo webhook". 
<br />
<div className="contenedor-imagenes">
  <ImageZoom
    src="/crearWebhook.webp"
    alt="Nuevo webhook"
    width={800}
    height={200}
  />
</div>
Al hacer clic en este botón, se abrirá el siguiente formulario que permite la entrada de datos para la creación de un nuevo permiso:

- **Método (Campo de selección, Obligatorio):** En este campo podrá seleccionar entre los diferentes métodos de petición, GET, POST, PUT y DELETE.
- **Tipo de evento (Campo de selección, Obligatorio):** En este campo podrá seleccionar entre los diferentes eventos que cuentan con esta funcionalidad los cuales son, crear cita, asistió a la cita y cancelar cita.
- **URL(Obligatorio):** Esta es la dirección URL donde se enviará la notificación. Es el endpoint que recibirá la información del evento.

Además, se pueden aplicar los siguientes filtros:

1. Filtro por **estado**: permite filtrar por Todos, Activos o Inactivos.
