# Gestionar citas 

La opción "Gestionar Citas" te brinda una herramienta centralizada para visualizar y administrar todas las citas creadas en tu sistema. Utilizando filtros como paciente, médico, servicio, sede, fecha, tipo y estado, puedes acceder a una lista detallada de citas que se ajustan a tus criterios de búsqueda. Para cada cita listada, tendrás acciones disponibles para gestionarlas según las necesidades, garantizando un seguimiento efectivo y una atención médica coordinada.

## Ingreso a la opción

Para acceder a la gestión de citas, debe hacer clic en la opción “Gestionar citas” en el menú principal del módulo de citas.

## Visualización de citas 

Dentro de la opción de "Gestión de citas", se encuentran disponibles los siguientes filtros para el listado de citas para interactuar con cada una de ellas:

- **Paciente:** Este campo filtrará la citas y solo listará las de un paciente especifico.
- **Médico:** Este campo filtrará la citas y solo listará las de un médico especifico.
- **Servicio: E**ste campo permite que solo se hagan visibles las citas en las que se presta un servicio especifico.   
- **Sede:** Este campo traerá los registros asociados a una de las sedes registradas en el sistema
- **Fecha:** El único campo obligatorio para empezar la búsqueda y consultar las citas que estés en el intervalo diligenciado.
- **Tipo:** En este campo se selecciona el tipo de cita que desea consultar, ya sea presencial, telemedicina o ambas
- **Estado:** Filtra las citas según su estado, que puede ser activa, cancelada, transferida, asignada, vencida o todas las opciones disponibles
