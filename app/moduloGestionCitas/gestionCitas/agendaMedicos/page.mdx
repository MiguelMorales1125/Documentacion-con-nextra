# Agenda médicos

Esta opción permite gestionar de manera eficiente las agendas de los médicos en tu empresa prestadora de salud. Con esta herramienta, puedes crear nuevas agendas, visualizar las existentes para coordinar citas, transferir entre médicos según sea necesario y cancelarlas si surge algún imprevisto. Esta funcionalidad te ofrece un control completo sobre la programación de espacios, garantizando una gestión ordenada y efectiva de los horarios médicos.

## Ingreso a la opción

Para acceder a la gestión de agendas, debe hacer clic en la opción “Agendas médicos” en el menú principal del módulo de citas.

## Visualización de agendas

Para la visualización y gestión de las agendas se cuenta con diferentes filtros según las necesidades, estos son:

- **Médico:** Permite buscar agendas asociadas a un médico específico utilizando su número de documento o nombre como criterio de búsqueda.
- **Servicio:** Filtra las agendas según el código o descripción del servicio al que están asignadas.
- **Sedes:** Permite buscar agendas asociadas a una sede específica utilizando su código o nombre como criterio de búsqueda.
- **Fecha:** Selecciona un intervalo de fechas para mostrar las agendas programadas dentro de ese período.
- **Tipo:** Permite seleccionar el tipo de agenda, ya sea presencial o telemedicina.
- **Estado:** Filtra las agendas según su estado, que puede ser activa, cancelada, transferida, asignada, vencida o todas las opciones disponibles.
