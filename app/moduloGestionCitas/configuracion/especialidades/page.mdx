# Especialidades

Las especialidades se refieren a las áreas específicas de la medicina en las que un médico está capacitado y tiene experiencia.

## Ingreso a la opción 
1. Para acceder, primero debe ingresar a la opción de configuración desde el menú principal del módulo de citas.
2.  Una vez en la configuración, se desplegará un submenú donde encontrará la opción específica de "Especialidades". Al seleccionar esta opción, podrá gestionar y administrar las especialidades de los médicos según las necesidades de su empresa prestadora de salud.

## Crear especialidad

Dentro de la opción de "Especialidades", se encuentra disponible un botón con el texto "Nueva especialidad". 
<br />
<div className="contenedor-imagenes">
  <ImageZoom
    src="/crearEspecialidad.webp"
    alt="Boton nueva especialidad"
    width={800}
    height={300}
  />
</div>

Al hacer clic en este botón, se abrirá el siguiente formulario que permite la entrada de datos para la creación de una nueva especialidad:

- **Código (Campo único)**: Este campo requiere un código único que identifique de manera exclusiva al grupo de servicio. El sistema verificará automáticamente la unicidad de este código para evitar duplicados y garantizar la integridad de los datos.
- **Descripción:** En este campo, se debe proporcionar una descripción concisa pero completa del grupo de servicio. Esta descripción ayudará a contextualizar y comprender mejor la función y el propósito del grupo de servicio dentro del sistema.

</div>
