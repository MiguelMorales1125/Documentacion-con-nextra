# Pacientes

Los pacientes son las personas que reciben atención médica y servicios de salud de los profesionales y establecimientos médicos. La información tratada en esta opción será compartida entre los módulos de  citas y digiturno.

## Ingreso a la opción 
1. Para acceder, primero debe ingresar a la opción de configuración desde el menú principal del módulo de citas.
2. Una vez en la configuración, se desplegará un submenú donde encontrará la opción específica "Pacientes". Al seleccionar esta opción, podrá gestionar y administrar los médicos disponibles en el sistema.

Al hacer clic en el botón "Nuevo médico" en la vista principal de la opción, se abre un formulario con los siguientes campos para la entrada de datos:

- **Tipo de Documento (Obligatorio, campo de selección):** Este campo ofrece una lista desplegable que enumera los tipos de documentos disponibles, como cédula de ciudadanía, tarjeta de identidad, pasaporte, etc.
- **Número de Identificación (Obligatorio):** En este campo se debe ingresar el número de identificación del médico.
- **Primer Nombre (Obligatorio):** Aquí se debe ingresar el primer nombre del médico.
- **Segundo Nombre:** Este campo es opcional y permite ingresar el segundo nombre del médico, si lo tiene.
- **Primer Apellido (Obligatorio):** Se debe ingresar el primer apellido del médico en este campo.
- **Segundo Apellido:** Este campo es opcional y permite ingresar el segundo apellido del médico, si lo tiene.
- **RH:** Campo opcional para ingresar el RH del paciente.
- **Sexo (Obligatorio, campo de selección):** Este campo ofrece la opción de seleccionar el sexo del médico, con las opciones "Masculino" y "Femenino".
- **Grupo Étnico (Obligatorio, campo de selección):** Aquí se debe seleccionar el grupo étnico al que pertenece el paciente. La lista desplegable mostrará los grupos étnicos disponibles para elegir.
- **Fecha de Nacimiento (Obligatorio):** Se debe ingresar la fecha de nacimiento del paciente en este campo.
- **Régimen (Obligatorio, campo de selección):** Este campo ofrece la opción de seleccionar el régimen al que pertenece el paciente, con las opciones "Contributivo" y "Subsidiado".
- **Ciudad de Residencia (Obligatorio, campo de selección):** Aquí se debe seleccionar la ciudad de residencia del paciente. La lista desplegable mostrará las ciudades disponibles para elegir.
- **Nivel Académico (Obligatorio, campo de selección):** En este campo se debe seleccionar el nivel académico del paciente, como preescolar, básica primaria, básica secundaria, etc.
- **Teléfono (Obligatorio):** Se debe ingresar el número de teléfono de contacto del médico.
- **Teléfono Secundario:** Este campo es opcional y permite ingresar un segundo número de teléfono del paciente, si lo tiene.
- **Dirección:** Campo obligatorio con la dirección de residencia del paciente.
- **Correo electrónico:** Campo obligatorio para ingresar el email actual del paciente.
- **Ocupación:** Campo opcional en el cual se filtrará y seleccionará la ocupación del paciente.[¿Como crear una ocupación?](/moduloGestionCitas/configuracion/ocupaciones)
- **Aseguradora** (Obligatorio, campo de búsqueda por código o nombre): Aquí se debe buscar y seleccionar la aseguradora del paciente. El campo permite realizar la búsqueda por código o nombre para encontrar la aseguradora correspondiente.
