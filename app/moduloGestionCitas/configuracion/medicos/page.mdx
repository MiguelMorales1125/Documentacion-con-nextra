# Médicos

Los médicos son profesionales de la salud capacitados para diagnosticar, tratar y gestionar la salud de los pacientes. Cada médico puede tener una o varias especialidades médicas en las que se especializa.

## Ingreso a la opción 
1. Para acceder, primero debe ingresar a la opción de configuración desde el menú principal del módulo de citas.
2. Una vez en la configuración, se desplegará un submenú donde encontrará la opción específica "Médicos". Al seleccionar esta opción, podrá gestionar y administrar los médicos disponibles en el sistema.

## Crear médicos

Dentro de la opción de "Médicos", se encuentra disponible un botón con el texto "Nuevo médico". 
<br />
<div className="contenedor-imagenes">
  <ImageZoom
    src="/nuevoMedico.webp"
    alt="Nuevo medico"
    width={800}
    height={300}
  />
</div>

Al hacer clic en el botón "Nuevo médico" en la vista principal de la opción, se abre un formulario con los siguientes campos para la entrada de datos:

- **Tipo de Documento (Obligatorio, campo de selección):** Este campo ofrece una lista desplegable que enumera los tipos de documentos disponibles, como cédula de ciudadanía, tarjeta de identidad, pasaporte, etc.
- **Número de Identificación (Obligatorio):** En este campo se debe ingresar el número de identificación del médico.
- **Primer Nombre (Obligatorio):** Aquí se debe ingresar el primer nombre del médico.
- **Segundo Nombre:** Este campo es opcional y permite ingresar el segundo nombre del médico, si lo tiene.
- **Primer Apellido (Obligatorio):** Se debe ingresar el primer apellido del médico en este campo.
- **Segundo Apellido:** Este campo es opcional y permite ingresar el segundo apellido del médico, si lo tiene.
- **Sexo (Obligatorio, campo de selección):** Este campo ofrece la opción de seleccionar el sexo del médico, con las opciones "Masculino" y "Femenino".
- **Fecha de Nacimiento (Obligatorio):** Aquí se debe ingresar la fecha de nacimiento del médico.
- **Teléfono (Obligatorio):** Se debe ingresar el número de teléfono de contacto del médico.
- **Registro médico (Obligatorio):** En este campo se ingresa el código del registro único nacional del talento humano en salud.
- **Correo Electrónico (Obligatorio):** En este campo se debe proporcionar la dirección de correo electrónico del médico.
