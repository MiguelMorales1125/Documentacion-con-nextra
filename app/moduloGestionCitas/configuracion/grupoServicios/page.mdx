# Grupos de servicio

Los Grupos de Servicio son conjuntos o categorías que agrupan varios tipos de servicios médicos relacionados entre sí.

## Ingreso a la opción 
1. Para acceder, primero debe ingresar a la opción de configuración desde el menú principal del módulo de citas.
2. Una vez en la configuración, se desplegará un submenú donde encontrará la opción específica de "Grupos de Servicio". Al seleccionar esta opción, podrá gestionar y administrar los grupos de servicios médicos según las necesidades.

## Crear grupo de servicio

Dentro de la opción de "Grupos de Servicio", se encuentra disponible un botón denominado "Nuevo grupo de servicio".
<br />
<div className="contenedor-imagenes">
  <ImageZoom
    src="/nuevoGrupoServicio.webp"
    alt="Nueva grupo servicio"
    width={800}
    height={300}
  />
</div>

Al hacer clic en este botón, se abrirá un formulario que permite la entrada de datos para la creación de un nuevo grupo de servicio. El formulario de ingreso de datos está diseñado para capturar información del grupo de servicio y consta de  los siguientes campos:

- **Código (Campo único/Obligatorio)**: Este campo requiere un código único que identifique de manera exclusiva al grupo de servicio. El sistema verificará automáticamente la unicidad de este código para evitar duplicados y garantizar la integridad de los datos.
- **Descripción (Obligatorio):** En este campo, se debe proporcionar una descripción concisa pero completa del grupo de servicio. Esta descripción ayudará a contextualizar y comprender mejor la función y el propósito del grupo de servicio dentro del sistema.
