# Creación de admin

- Para empezar la configuración de un **nuevo cliente** primero se crea un administrador con todos los permisos, este registro se insertará por medio de Postman.

![Creacion de un usuario administrador](/CreacionAdmin.webp)

Este administrador contará con todos los permisos de acceso y acciones para todas las funcionalidades del aplicativo, además no estará asociado a ninguna empresa.
